\# Online Course Assessment Application



\## 📌 Project Overview

This is a Django-based web application that allows users to take course assessments and view their results.



\## 🚀 Features

\- Course creation using admin panel

\- Add questions and multiple choices

\- Select answers and submit exam

\- Automatic score calculation

\- Result display



\## 🛠️ Tech Stack

\- Python

\- Django

\- HTML (Templates)



\## ▶️ How to Run

1\. Activate virtual environment:

&#x20;  venv\\Scripts\\activate



2\. Run server:

&#x20;  python manage.py runserver



3\. Open browser:

&#x20;  http://127.0.0.1:8000/
